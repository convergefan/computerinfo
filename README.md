# computerinfo

Small tool made in order to learn how to utilise libraries. 

The tool shows hardware information.

# Prerequisites

$ pip install py-cpuinfo

$ pip install psutil


# Installing

Download the py or clone the repo. Run the file: python3 computerinfo.py
